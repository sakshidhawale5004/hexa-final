## 📁 **Why Double hexatp-main?**

Your directory structure is:
```
C:\xampp3\htdocs\
└── hexatp-main\          ← First level
    └── hexatp-main\      ← Second level (your files are here)
        ├── index.html
        ├── contact.html
        ├── check_status.php
        └── ... (all other files)
```

So the URL needs both levels:
```
http://localhost/hexatp-main/hexatp-main/filename.php
                 └─────┬─────┘ └─────┬─────┘
                   First        Second
                   level        level
```

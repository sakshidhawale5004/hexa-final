## ➕ Adding a New Country

### Step-by-Step Process

#### 1. Hero Section
- **Country Name**: Full country name (e.g., "New Zealand")
- **Country Code**: 2-letter code (e.g., "NZ")
- **Flag URL**: Link to country flag image
- **Hero Title**: Main heading (e.g., "Transfer Pricing New Zealand")
- **Hero Description**: Brief introduction (2-3 sentences)

#### 2. SEO Meta Tags
- **Meta Title**: Max 60 characters (appears in search results)
- **Meta Description**: Max 160 characters (search snippet)

#### 3. Overview Section
- **Left Column**: First half of overview content
- **Right Column**: Second half of overview content
- **Formatting**: Use toolbar for bold, italic, lists, links

#### 4. Regulatory Frameworks (3 Boxes)
- **Box 1**: Primary legislation
- **Box 2**: Documentation requirements
- **Box 3**: Penalties/compliance
- Each box has: Title + Description

#### 5. Documentation Cards
- **Title**: What users see before expanding
- **Short Description**: Brief summary
- **Detailed Content**: Full information with formatting
- **Actions**: Add, reorder (drag-drop), delete

## 💾 Saving Your Work

| Button | Action | When to Use |
|--------|--------|-------------|
| **Save Draft** | Saves without publishing | Work in progress, needs approval |
| **Preview** | Shows how it will look | Before publishing |
| **Publish** | Makes content live | Ready for public viewing |

## ✏️ Editing Existing Content

### From Countries List:
1. Click **"Countries List"** in navigation
2. Find your country (use search if needed)
3. Click **"Edit"** button
4. Make your changes
5. Click **"Publish"** to update

### Quick Actions:
- **Edit**: Update content
- **View**: See live page
- **Delete**: Remove country (with confirmation)

## 🔍 Finding Content

### Search
- Use search box in Countries List

### Filter
- **All**: Show everything

- **Lists**: Organize information
  - Bullet points for items
  - Numbered lists for steps

✅ Keep hero descriptions concise (2-3 sentences)

## 🆘 Troubleshooting

### Issue: Can't login
**Solution**: Contact your administrator for credentials

### Issue: Changes not showing
**Solution**: 
1. Make sure you clicked "Publish" (not just "Save Draft")
2. Clear your browser cache
3. Refresh the page

### Issue: Lost my work
**Solution**: 
- Drafts are auto-saved every 2 minutes
- Check "Drafts" filter in Countries List

### Issue: Can't delete a country
**Solution**: Only administrators can delete. Contact your admin.

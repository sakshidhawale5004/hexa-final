🔬 Root Cause Hypothesis

Based on the design document, the most likely root causes are:

  1. Font Rendering Failure: Poppins font may not load properly on mobile devices
  2. CSS Specificity Issues: Mobile CSS rules may override logo styles unexpectedly
  3. Character Encoding Issues: Text content may have encoding problems on mobile browsers
  4. Font Weight/Style Mismatch: font-weight: 800 may not be supported on mobile

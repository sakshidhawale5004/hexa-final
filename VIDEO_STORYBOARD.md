## 🎨 VISUAL ELEMENTS LEGEND

### Symbols Used
- `┌─┐ └─┘` = Boxes/containers
- `[Button]` = Clickable button
- `[____]` = Input field
- `▼` = Dropdown/expandable
- `✓` = Checkmark/completed
- `⋮⋮` = Drag handle
- `←` = Arrow/pointer
- `🔒 👤 📊 🌍 ➕ 🚪` = Icons

### Color Coding (for production)
- **Yellow (#f5c400)**: Highlights, active states, callouts
- **Green (#10b981)**: Success messages, publish actions
- **Red (#ef4444)**: Delete actions, warnings
- **Blue (#3b82f6)**: Links, info
- **Gray (#94a3b8)**: Secondary text, borders

### Animation Types
- **Fade**: Smooth opacity transition
- **Slide**: Element moves from off-screen
- **Glow**: Pulsing highlight effect
- **Type**: Text appears character by character
- **Zoom**: Scale in/out effect
- **Pulse**: Rhythmic size change

- **Resolution**: 1920x1080 (Full HD)

- **Callout Placement**: Avoid corners

## 🎬 PRODUCTION NOTES

### Camera Movements
- **Pan**: Slow horizontal scroll
- **Tilt**: Vertical scroll through forms
- **Zoom**: Focus on specific elements
- **Hold**: Static shot for reading

### Timing Guidelines
- **Reading Time**: 3 seconds per sentence
- **Action Time**: 2 seconds per click
- **Transition Time**: 0.5-1 second
- **Pause Time**: 1 second between sections

### Cursor Behavior
- **Speed**: Moderate (not too fast)
- **Highlight**: Yellow spotlight follows cursor
- **Clicks**: Brief flash on click
- **Hover**: Slight pause before clicking

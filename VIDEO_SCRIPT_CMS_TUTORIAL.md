## 📹 RECORDING TIPS

### Screen Recording
1. **Resolution**: 1920x1080 (Full HD)
2. **Frame Rate**: 30 fps minimum
3. **Browser**: Use Chrome in clean profile (no extensions visible)
4. **Window Size**: Maximize browser window
5. **Cursor**: Use cursor highlighting software

### Audio Recording
1. **Microphone**: Use quality USB microphone
2. **Environment**: Quiet room, no background noise
3. **Format**: WAV or high-quality MP3 (320kbps)
4. **Levels**: Check audio levels before recording

### Editing
1. **Software**: Adobe Premiere Pro, Final Cut Pro, or DaVinci Resolve
2. **Cuts**: Remove mistakes, long pauses, loading times
3. **Speed**: Speed up repetitive actions (1.5x-2x)
4. **Music**: Add subtle background music (low volume)
5. **Captions**: Add closed captions for accessibility

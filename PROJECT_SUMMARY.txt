                         DATABASE SCHEMA
================================================================================

TABLE: consultations
├── id (INT, Primary Key, Auto Increment)
├── name (VARCHAR 100)
├── email (VARCHAR 100)
├── phone (VARCHAR 20)
├── consultation_type (VARCHAR 100)
├── appointment_date (DATE)
├── appointment_time (VARCHAR 20)
├── message (LONGTEXT)
├── status (ENUM: pending, confirmed, completed, cancelled)
├── created_at (TIMESTAMP)
├── updated_at (TIMESTAMP)
└── Indexes: email, date, status

TABLE: inquiries (Backward Compatible)
├── id (INT, Primary Key, Auto Increment)
├── name (VARCHAR 100)
├── email (VARCHAR 100)
├── phone (VARCHAR 20)
├── message (LONGTEXT)
├── created_at (TIMESTAMP)
└── Index: email

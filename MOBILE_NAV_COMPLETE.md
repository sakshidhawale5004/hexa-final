## 🎨 **CUSTOMIZATION (Optional)**

If you want to customize the mobile menu:

### **Change Menu Width:**
Find this in the CSS:
```css
.mobile-menu {
    width: 280px; /* Change to 320px or any width */
}
```

### **Change Colors:**
```css
.mobile-menu {
    background: rgba(5, 10, 20, 0.98); /* Menu background */
}

.mobile-menu ul li a:hover {
    color: #f5c400; /* Hover color */
}
```

### **Remove Emojis:**
Edit the HTML section and remove emojis:
```html
<!-- Before -->
<li><a href="index.html">🏠 Home</a></li>

<!-- After -->
<li><a href="index.html">Home</a></li>
```

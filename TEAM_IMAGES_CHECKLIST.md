## 📞 TROUBLESHOOTING

### If images don't show:
1. Check file names match exactly (case-sensitive!)
2. Verify files are in `public_html/` root (not in a subfolder)
3. Clear browser cache (Ctrl+F5)
4. Check file permissions (should be 644)

### If images are low quality:
1. Verify you uploaded the "new" versions
2. Check file sizes (should be reasonable, not tiny)
3. Don't use browser zoom - test at 100%

### If mobile doesn't work:
1. Clear mobile browser cache
2. Test in different browsers
3. Check responsive CSS is loaded
